**Entity Annotator - KA Project 2020**
-
**Requirements**
-
- python >=3.6.X
- pip3

**Installation**
-
- `pip install -r requirement.txt`
- `python nltk_download.py`

**Run the project**
-
- `python manage.py runserver`
- go to http://localhost:8000/

**Train the model**
- `cd model_trainer/`
- choose the amount of sentences in the code (`df = df[:<amount>]`)
- `python train.py`
- wait
